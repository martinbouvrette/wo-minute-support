# Wo-Minute — Support

**Wo-Minute** est un minuteur visuel pour iPhone et iPad : le temps ne s'affiche pas, il se **regarde**. Une seconde = un mouvement.

## Besoin d'aide ?

Pour toute question, suggestion ou problème : support@wominute.com

J'essaie de répondre en quelques jours.

## Questions fréquentes

**Le minuteur sonne-t-il si l'écran est éteint ?**
Oui, si vous avez autorisé les notifications au premier lancement. Sinon : Réglages ▸ Wo-Minute ▸ Notifications.

**Pas de son dans le simulateur de mes écouteurs ?**
Vérifiez le bouton silencieux de l'appareil — les sons d'alerte le respectent.

**La voix du minuteur Robot peut être plus belle.**
Téléchargez une voix française de qualité supérieure dans Réglages ▸ Accessibilité ▸ Contenu énoncé ▸ Voix ▸ Français (par exemple une voix « Premium »). Wo-Minute la choisira automatiquement.
